# 🌍 Travel Planner App  

## 📖 Overview  
A **travel planning web app** that fetches real-time travel details using **APIs** to provide users with:  
✅ Location details (GeoNames API)  
✅ 7-day weather forecast (Weatherbit API)  
✅ Destination images (Pixabay API)
